# 🚀 ASA Performance Boost – FPS Optimizer for ARK: Survival Ascended

---

## 📌 About

This tool is designed to improve performance in ARK: Survival Ascended. It applies safe, tested tweaks to the game's configuration files and system settings, resulting in higher FPS and smoother gameplay – especially on mid‑range and older hardware.

What it does:  
✅ Optimizes Engine.ini with UE5 performance parameters  
✅ Reduces stuttering and input lag  
✅ Adjusts background process priority (optional)  
✅ Easy one‑click restore to default settings

---

## ⚙️ How to Use

1. Download the latest release or clone this repository.
2. Right‑click optimize.bat and select Run as administrator (recommended for best results).
3. Choose an option from the menu:
   - 1 – Apply performance tweaks  
   - 2 – Restore original settings  
   - 3 – Exit

The script automatically backs up your original Engine.ini before making changes.

> 🔧 Manual tweaking: If you prefer, you can edit config/Engine.ini yourself and copy it to  
> %LOCALAPPDATA%\ARKSurvivalAscended\Saved\Config\WindowsClient\Engine.ini

---

## 🛡️ Important Notes

- This tool does not modify game binaries and will not trigger anti‑cheat.
- All changes are reversible via the "restore" option.
- No third‑party injectors or external processes are used.
